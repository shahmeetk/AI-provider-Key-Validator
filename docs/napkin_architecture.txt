# LLM API Key Validator Architecture

[User] -> [User Interface Layer]
[User Interface Layer] -> [Business Logic Layer]
[Business Logic Layer] -> [Data Access Layer]
[Data Access Layer] -> [External Systems]

[User Interface Layer] {
  [Single Key Validation]
  [Bulk Validation]
  [Validation History]
  [Provider Info]
  [Settings & Configuration]
}

[Business Logic Layer] {
  [Key Validators]
  [Provider Detection]
  [Validation Storage]
  [CSV Processing]
  [Error Handling]
}

[Data Access Layer] {
  [Provider Metadata]
  [History Storage]
  [Local File I/O]
  [Logging System]
  [External API Clients]
}

[External Systems] {
  [OpenAI API]
  [Claude API]
  [Mistral API]
  [Groq API]
  [Other LLM APIs]
}

# Data Flow Paths
[User] -> [Single Key Validation] -> [Key Validators] -> [External API Clients] -> [OpenAI API]
[User] -> [Bulk Validation] -> [CSV Processing] -> [Key Validators] -> [External API Clients]
[Key Validators] -> [Validation Storage] -> [History Storage]
[Provider Detection] -> [Provider Metadata]
[Error Handling] -> [Logging System]
